- chemistry-quick-ref.md — targets, dosing, formulas

- maintenance-procedures.md — drain/refill, filter service, winterizing, vacation mode

- Parts diagrams and visual references

- User asks about warranty coverage or claim process

- Need site prep measurements (pad dimensions, access clearances)
- Need visual diagram of component location
